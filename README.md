## About

CodeCraft Innovations serves as a digital portfolio, highlighting the developer's creative work and skills through a visually dynamic and futuristic design.

## Features

- Personal profile section
- Glitch effect header title
- Showcase of individual projects with clickable links
- Responsive background image and layout

## Tech Stack

- **HTML**: Structure of the webpage.
- **CSS**: Styling with cyberpunk themes and responsive design.
- **JavaScript**: Interactive elements and glitch effects.
- **Google Fonts API**: Custom typography for aesthetic appeal.

## Usage

To run this project locally:

1. Clone the repository:  
   `git clone https://github.com/dragonpilee/codecraft-innovations.git`

2. Navigate to the project folder and open the `index.html` file in your web browser.
